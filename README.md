# Quran0

Quran0 is a Quran website scaffolded with TanStack Start.

## Stack

- TanStack Start + TanStack Router
- React 19
- Tailwind CSS 4

## Local Development

```bash
npm install
npm run dev
```

The dev server runs on [http://localhost:3000](http://localhost:3000).

## Project Notes

- No backend, authentication, or database is configured.
- Quran API integration, progress storage, and PWA/offline support will be added
  later step by step.
